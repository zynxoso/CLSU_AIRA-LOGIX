# 🚀 AIRA-LOGIX

> AI-powered research intelligence, ICT analytics, and financial management platform built with Laravel 12, React 19, and Inertia.js.

AIRA-LOGIX is a modern full-stack platform designed to streamline ICT operations, automate research workflows, and deliver actionable analytics through intelligent data processing. Powered by Gemini AI and Neuron AI, the system transforms documents, spreadsheets, and images into structured insights and comprehensive reports.

## ✨ Features

* 📊 ICT analytics and service request monitoring
* 🧠 AI-powered research intelligence and data extraction
* 💸 Financial and expense management modules
* 📄 Export reports to Excel, CSV, DOCX, and PDF
* 🌗 Responsive dashboard with light/dark mode support
* ⚡ High-performance Laravel + React architecture

## 🛠️ Tech Stack

* **Backend:** Laravel 12, PHP 8.2+
* **Frontend:** React 19, Inertia.js 2, Tailwind CSS 4
* **AI Integration:** Gemini AI, Neuron AI
* **Visualization:** Recharts, Mermaid.js
* **UI:** Radix UI, Lucide React
* **DevOps:** Docker (Laravel Sail), Vite, PHPUnit

## 🚀 Quick Start

```bash
git clone https://github.com/zynxoso/CLSU_AIRA-LOGIX.git

composer install
npm install

cp .env.example .env
php artisan key:generate

php artisan dev
```

## 📦 Production Build

```bash
npm run build
php artisan migrate --force
```
